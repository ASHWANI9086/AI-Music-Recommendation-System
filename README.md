# 🎧 AI Music Recommendation System

An ML-powered music recommendation system using Spotify API and Streamlit.

## 🚀 Features
- Song recommendations
- Spotify integration
- Album images
- Audio preview
- Premium UI

## 🛠 Technologies
- Python
- Streamlit
- Scikit-learn
- Spotify API
- Pandas

## ▶ Run

pip install -r requirements.txt

python train.py

streamlit run app.py
